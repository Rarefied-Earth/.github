# Rarefied Earth

**Engineering-grade AI for operating companies.**

Rarefied Earth is an AI-systems implementation firm for operating SMBs, run with civil-engineering discipline. The firm builds AI-augmented workflow automation, agent systems, and the operating substrate companies run on. Florida S-corporation (Earth Evocation Inc. d/b/a Rarefied Earth), founded 2018, based in St. Petersburg.

## Three layers, one identity

1. **AI-systems implementation** for the companies that actually run things: workflow automation, document intelligence, multi-agent orchestration, custom internal portals.
2. **Civil-engineering depth** where it counts: the founder is a Florida Engineer Intern, MSCE (structural), and a coauthor of peer-reviewed prestressed-concrete restoration research in *PCI Journal*. In heavy-civil and construction-adjacent verticals, the credentials are the trust wedge.
3. **Build the Company**, the productized layer: the operating substrate Rarefied Earth runs on, packaged as a deployable engagement. Twenty canonical modules, field-tested on the firm itself before any client installation.

## How the work runs

- **Dogfood first.** Every system ships to a client only after it has run in production on Rarefied Earth's own pipeline. Clients are the second installation, not the first to hit a bug.
- **Human-in-the-loop.** AI drafts; humans approve. Every irreversible action sits behind a human gate.
- **No lock-in.** Python, PostgreSQL, FastAPI, standard APIs. Any competent engineer can pick the work up later.
